Meu Portfólio

No meu portfólio, eu não foquei apenas na interface. Desenvolvi uma estrutura em Javascript que simula o comportamento de um sistema distribuido, com barras de carregamento que representam uma latência de uma rede IP e o tempo de resposta de um servidor em Java com um fluxo de dados do cliente ao servidor.

Projetos desenvolvidos com foco em soluções de suporte e eficiência de processamento. Aqui você encontrará aplicações que unem o código Java com a agilidade do desenvolvimento Web, sempre priorizando o monitoramento de perfomance e a facilidade de suporte técnico em TI.

Cada projeto aqui seguiu um levantamento de requisitos e testes, garantindo que o software suporte as demandas de uma rede de sistemas real.
